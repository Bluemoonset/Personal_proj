Technical Projects

Slickdeal scraper

A Python based web scraper for Slickdeals that would collect data from a search and provide the user with data such as the average
price for the search item or what time of year would be most likely to have the best deals. Data would be stored
inside Excel files and were readily available to view in the form of charts. One can also manually manipulate the data into
custom graphics thanks to this storage of information. Unfortunately, Slickdeals has changed its UI
recently, and the current iteration of this project will not work on the new site. An update is in progress.

Custom linux terminal

A terminal written in C++ for an operating systems class. This terminal replicated the functions of commonly used terminals like
Ubuntu, being able to run commands, change directories, or even generate keys. Creating a terminal such as this requires consideration
for every command possible, and is not feasible for one person to create a professional-level clone. Thus, it was designed only to perform all
common commands, but does not to work for every command.

Planned Future Projects

A project that I have wanted to start for a long while is a depth perception camera. I originally planned to have
two cameras placed on mounts that could move left and right which, combined with the same-time viewing from the cameras,
would allow an accurate detection of distance from the camera. If this project were to succeed, I'd like to expand upon it
by integrating it into a phone plugin with its own app; the device would plug into the phone through the charger port
and then the app would display a camera and some text, where the text would show the distance of any object pointed to by the camera.
